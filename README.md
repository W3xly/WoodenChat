# WoodenChat

Jednoduchá chatovací aplikace.

Aplikací napsaná ve Swift 5 - bez užití storyboards. <br>
Jako backend využívá Firebase Firestore. <br>
Při vývoji jsem využil MVVM architekturu. <br>
